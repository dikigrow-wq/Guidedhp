DHP Creative Plan System

Panduan lengkap memahami sistem manajemen internal DHP Creative — dari alur login, peran tim, hingga cara kerja setiap modul.

Mulai Membaca
Tentang Sistem

Satu platform,
banyak peran

DHP Creative Plan System adalah platform manajemen terpusat yang menghubungkan seluruh tim — mulai dari sales, host live, marketing, hingga manajemen puncak — dalam satu ekosistem yang terstruktur dan real-time.

01
Multi-Role Access
Enam level akses berbeda — setiap peran melihat dan mengelola data yang relevan dengan tugasnya saja.
02
Data Real-Time
Semua data tersinkronisasi langsung via cloud. Tidak ada delay, tidak ada data ganda, selalu up-to-date.
03
Dashboard & Analitik
Visualisasi performa tim secara menyeluruh — dari lead pipeline, GMV host live, hingga efektivitas budget marketing.
04
Notifikasi Sistem
Pemberitahuan otomatis untuk pengingat laporan, update status, dan deadline yang perlu diperhatikan.
Cara Masuk

Alur Login
Step by Step

Sistem menggunakan autentikasi berbasis peran. Setiap anggota tim masuk sesuai dengan tanggung jawabnya masing-masing.

1
Langkah Pertama
Buka Platform
Akses DHP Creative Plan System melalui browser. Setelah loading selesai, halaman login akan tampil otomatis dengan tampilan elegan bertema gelap. Tidak perlu membuat akun — sistem menggunakan akun terpusat per peran.
2
Langkah Kedua
Pilih Peran Anda
Pilih salah satu dari enam peran yang tersedia: Moderator, Admin, B2B, Host Live, Marketing, atau Guest. Klik kartu peran sesuai posisi Anda di tim. Kartu akan aktif (bercahaya) sebagai konfirmasi pilihan.
Moderator Admin B2B Host Marketing Guest
3
Langkah Ketiga
Masukkan Password
Setelah memilih peran, halaman bergeser ke form autentikasi. Masukkan password yang telah diberikan oleh admin sistem. Password bersifat rahasia dan tidak dibagikan ke publik. Untuk role Host Live, pilih dulu nama akun (Nina / Sekar) sebelum memasukkan password.
Keamanan: Jangan bagikan password kepada siapapun di luar tim. Setiap peran memiliki akses berbeda yang telah disesuaikan.
4
Langkah Keempat
Masuk ke Dashboard
Login berhasil — sistem menyambut Anda dengan pesan inspiratif yang dipersonalisasi. Dashboard langsung terbuka menampilkan data dan fitur sesuai peran Anda. Sidebar navigasi di sebelah kiri menjadi panduan utama berpindah antar modul.
✓
Mode Khusus
Guest — Tanpa Password
Role Guest tidak memerlukan password. Klik "Lihat Dashboard" untuk langsung masuk dalam mode baca. Guest dapat melihat analitik, data tim, dan dashboard marketing — tetapi tidak dapat membuat, mengedit, atau menghapus data apapun.
Struktur Tim

Enam Peran,
Satu Tujuan

Setiap peran dirancang dengan akses dan tanggung jawab yang spesifik agar alur kerja tim tetap terstruktur dan efisien.

Moderator
Akses Penuh
Level tertinggi dalam sistem. Moderator memiliki visibilitas dan kendali penuh atas seluruh data tim, dapat mengatur jadwal, menyetujui laporan, dan mengirim notifikasi ke semua anggota.
  • Dashboard semua role secara bersamaan
  • Kelola kalender iklan & jadwal live host
  • ACC & validasi laporan harian host
  • Update target bulanan host live
  • Lihat & kelola semua data leads
  • Kirim notifikasi ke seluruh tim
  • Akses dashboard marketing read-only
Admin
Sales Internal
Mengelola leads dan pipeline penjualan untuk jalur internal. Admin fokus pada data customers, kalender leads, analitik performa, dan input laporan harian aktivitas mereka.
  • Dashboard personal dengan KPI hari ini
  • Kalender leads — lihat & buat jadwal
  • Analitik performa sales admin
  • Tabel data customers lengkap
  • Input laporan harian kegiatan
B2B
Business-to-Business
Khusus mengelola relasi dan pipeline kerjasama business-to-business. Memiliki sistem leads, kalender, dan analitik yang terpisah dari jalur admin untuk menjaga fokus.
  • Dashboard B2B dengan progress pipeline
  • Kalender leads B2B tersendiri
  • Analitik konversi & performa B2B
  • Tabel customers B2B dengan status deal
  • Input laporan harian aktivitas B2B
Host Live
Nina / Sekar
Dua akun host tersedia: Nina dan Sekar. Masing-masing memiliki target GMV bulanan sendiri. Host dapat melihat jadwal live, mencatat laporan per sesi, dan memantau progress menuju target.
  • Dashboard performa live dengan visual target
  • Kalender jadwal live bulanan
  • Input laporan per sesi live (GMV, durasi)
  • Pantau progress target bulanan real-time
  • Kutipan motivasi personal setiap sesi
Marketing
Strategi & Budget
Mengelola ekosistem marketing secara menyeluruh — dari perencanaan anggaran bulanan hingga tracking performa KOL & affiliate yang digunakan dalam kampanye produk.
  • Dashboard marketing dengan ringkasan bulanan
  • Kelola budget marketing per periode
  • Input & track KOL/affiliate per kampanye
  • Monitor ERF (Engagement Rate Factor) tiap KOL
  • Lacak progress per status konten
Guest
Read Only
Mode tamu tanpa password. Ideal untuk stakeholder, mitra, atau pihak yang perlu memantau data tanpa hak edit. Akses terbatas namun informatif.
  • Analitik semua role dalam satu halaman
  • Pantau semua task tim lintas divisi
  • Dashboard marketing dalam mode baca
  • Tidak dapat membuat atau mengubah data

Cara Kerja

Workflow
per Divisi

Pelajari alur kerja harian masing-masing divisi dalam sistem — dari input data hingga laporan akhir.

01
Tambah Lead Baru
Masukkan data customer baru ke dalam sistem — nama, kontak, produk yang diminati, dan sumber lead.
02
Jadwalkan Follow-Up
Set jadwal follow-up di kalender leads. Sistem akan menampilkan reminder sesuai tanggal yang ditentukan.
03
Update Status
Setelah interaksi, perbarui status lead: New → Follow-Up → Closing → Deal atau Blocked sesuai perkembangan.
04
Input Laporan Harian
Di akhir hari, isi laporan aktivitas sales harian — jumlah kontак, follow-up dilakukan, dan hasil yang didapat.
05
Pantau Analitik
Lihat grafik performa mingguan dan bulanan di halaman analitik — konversi rate, status distribusi, tren leads.
Status Lead yang Tersedia:
Baru Masuk Follow-Up Closing Deal Selesai Diblokir Reschedule
01
Identifikasi Prospek B2B
Catat perusahaan atau brand yang berpotensi untuk diajak kerjasama — lengkap dengan PIC dan info kontak.
02
Approach & Negosiasi
Lakukan pendekatan awal, presentasi, dan negosiasi. Update status di sistem sesuai tahap yang dicapai.
03
Closing Deal
Tandai status sebagai "Closing" saat memasuki tahap final. Dokumentasikan nilai deal dan catatan penting.
04
Laporan Harian B2B
Input aktivitas B2B harian — jumlah perusahaan dihubungi, pertemuan berlangsung, dan prospek aktif saat ini.
05
Evaluasi Pipeline
Gunakan halaman analitik B2B untuk mengevaluasi konversi pipeline dan mengidentifikasi hambatan dalam proses.
01
Cek Jadwal Live
Buka halaman kalender untuk melihat jadwal live hari ini — jumlah sesi, jam mulai, dan durasi yang direncanakan.
02
Jalankan Sesi Live
Lakukan sesi live sesuai jadwal. Catat GMV yang tercapai, jumlah viewer, dan durasi aktual sesi.
03
Input Laporan Sesi
Masukkan data per sesi di halaman "Input Report Harian" — GMV, waktu mulai & selesai, dan catatan jika ada kendala.
04
Deadline Laporan
Perhatikan deadline pengisian laporan. Sistem akan memberikan peringatan visual jika laporan belum diisi mendekati batas waktu.
05
Pantau Target Bulanan
Halaman target menampilkan progress GMV secara visual — persentase pencapaian dan jarak menuju target bulan ini.
Perayaan Target: Saat target bulanan tercapai, sistem menampilkan animasi perayaan khusus sebagai apresiasi atas kerja keras host. 🏆
01
Set Budget Bulanan
Awal bulan, tentukan total alokasi anggaran marketing. Budget ini menjadi acuan untuk semua aktivitas di bulan tersebut.
02
Input Data KOL
Tambahkan data KOL atau afiliasi yang digunakan — nama, tier (nano/micro/macro), platform, produk, dan metode pembayaran.
03
Track Progress Konten
Update status konten per KOL: Sudah Tektokan → Deal → Produk Dikirim → Konten Dibuat → Upload → Selesai.
04
Input ERF & Rating
Setelah konten tayang, input Engagement Rate Factor (Instagram & TikTok) dan berikan rating performa KOL tersebut.
05
Evaluasi Dashboard
Pantau dashboard marketing untuk melihat realisasi vs budget, distribusi pengeluaran, dan efektivitas kolaborasi KOL.
01
Monitor Dashboard
Pantau dashboard semua role dalam satu tampilan — performa admin, B2B, host, dan marketing secara bersamaan.
02
Atur Jadwal Host
Buat dan kelola jadwal live untuk Nina dan Sekar per bulan — tentukan hari, jumlah sesi, dan tandai hari libur.
03
Validasi Laporan Host
Review dan setujui laporan harian yang disubmit host. Moderator dapat membuka laporan yang terkunci jika diperlukan koreksi.
04
Update Target
Set atau perbarui target GMV bulanan untuk masing-masing host sesuai kondisi dan kapasitas tim.
05
Kirim Notifikasi
Buat dan kirim pengumuman atau reminder ke seluruh tim melalui modul notifikasi. Semua user menerima secara real-time.
Best Practice

Tips Penggunaan
Terbaik

Panduan singkat agar pengalaman menggunakan sistem ini maksimal dan data tim selalu akurat.

Input Laporan di Hari yang Sama
Selalu masukkan laporan harian di hari yang sama dengan aktivitas. Laporan yang terlambat bisa terkunci dan memerlukan approval moderator untuk dibuka.
Update Status Lead Segera
Perbarui status lead segera setelah ada interaksi. Data yang akurat menghasilkan analitik yang dapat dipercaya untuk pengambilan keputusan.
Aktifkan Panel Notifikasi
Cek notifikasi secara rutin dengan klik ikon lonceng di pojok kanan atas. Notifikasi penting dari moderator bisa berisi instruksi atau update sistem.
Akses dari Mobile Friendly
Sistem responsif dan dapat diakses dari perangkat mobile. Gunakan tombol menu (☰) di pojok kiri atas untuk membuka sidebar navigasi saat di HP.
Jaga Kerahasiaan Password
Password setiap peran bersifat confidential. Jangan bagikan ke pihak luar tim. Jika ada dugaan kebocoran, segera hubungi admin sistem.
Butuh Bantuan Teknis?
Jika mengalami kendala teknis, gunakan tombol "Bantuan" di pojok kanan atas untuk langsung terhubung ke dukungan teknis DHP Creative via WhatsApp.
Sistem ini dirancang untuk mendukung, bukan memperumit. Setiap fitur hadir karena ada kebutuhan nyata dari tim. Jika ada saran peningkatan atau fitur yang dibutuhkan, sampaikan melalui jalur komunikasi internal DHP Creative.
